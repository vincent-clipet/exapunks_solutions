**DA**

; FILE DATA SENDER

LINK 800

MARK LOOP

@REP 9

**PR**

```
; PROCESSING

NOOP
LINK 800


; CALCULATIONS
MARK LOOP
GRAB M
SEEK 1
ADDI F F X
ADDI X F X
DIVI X 3 T;(BA+ZA+APB)/3
REPL LOOP
MULI F F X
DIVI X F X;(WRT*OI)/OD
ADDI X T T
SUBI F F X
MULI X 20 X;(PC-PS)*20


; SEND
MODE ; GLOBAL TO HS
SEEK -9999
ADDI X T M ; SEND SCORE
COPY F M   ; SEND NAME
ADDI X T M ; SEND SCORE
```
